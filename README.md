# M133-Dorfladen

## Built with
* [NodeJS](https://nodejs.org/) - v10.16.3
* [NPM](https://www.npmjs.com/) - v6.13.4

## Getting Started
### Installation
First all dependencies have to be installed:
```
npm install
```
Next the application can be configured as follows:
```
npm start
```
